# Web Frontend

React web application for AR Diagram Viewer.

## Development
```bash
npm start
```

Runs on http://localhost:3000

## Build
```bash
npm run build
```

Creates optimized production build in `build/`

## Features

- Drag & drop file upload
- Interactive AR overlays on images
- Component detection and listing
- Real-time AI chat with document context
- Responsive 3-panel layout
