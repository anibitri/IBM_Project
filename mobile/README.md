# Mobile Frontend

React Native mobile app (Expo) for AR Diagram Viewer.

## Development
```bash
npx expo start
```

Then:
- Press `a` for Android emulator
- Press `i` for iOS simulator
- Scan QR code with Expo Go app (physical device)

## Features

- Native file picker integration
- AR overlays using React Native SVG
- Native navigation
- Real-time AI chat
- Component detection visualization

## Configuration

Edit `app.json` for app settings (name, icons, splash screen, etc.)
